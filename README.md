# BankSys - Sistema de Gestión Financiera

Este proyecto consiste en una aplicación desarrollada en Python utilizando Tkinter para la gestión de servicios financieros de una entidad llamada "Inversiones Seguras S.A.".

El sistema permite visualizar formularios para la administración de información relacionada con clientes, cuentas bancarias, créditos y transacciones financieras.

## Tecnologías utilizadas
- Python
- Tkinter
- Programación orientada a objetos
- GitHub

## Módulos del sistema
El sistema está dividido en cuatro módulos principales:

1. Clientes
2. Cuentas bancarias
3. Créditos
4. Transacciones financieras
